# Simple Nodejs HTTP Server Example

This example demonstrates a basic HTTP server built with Node.js. It showcases Node.js's ability to create server-side applications and respond to web requests using its core `http` module. This is a common 'hello world' type of application after successfully installing Node.js.

## Language

`javascript`

## How to Run

1. Save the code as `server.js`.
2. Open your terminal, navigate to the file's directory, and run `node server.js`.
3. Open your web browser and go to `http://localhost:3000` to see the output.
